# Password Sharing Web App

A secure web application for sharing passwords. Once a password is retrieved, it is permanently deleted from the database.

## Features
- Generate shareable links for passwords.
- Passwords are encrypted and deleted after retrieval.
- Passwords expire after 7 days if unused.
- Simple and user-friendly interface.

## Setup
1. Clone the repository.
2. Create a `config.php` file with your database credentials and encryption key.
3. Deploy to your server.
